# 12-Week Engineering Excellence Roadmap

## Philosophy

**Week 1-4:** Pattern Foundation (DSA Mental Models)
**Week 5-8:** Pattern Mastery + System Design Introduction
**Week 9-10:** Advanced Problems + Full System Design
**Week 11-12:** Interview Simulation Intensive

---

## Month 1: Pattern Recognition (Weeks 1-4)

### Week 1: Array & String Mastery

**DSA Patterns:**

- Two Pointers
- Sliding Window
- Prefix Sum

**Daily Flow:**

- Morning: 2 problems (1 easy, 1 medium)
- Evening: Pattern study + add to library
- Focus: Recognition speed over solution speed

**System Design:**

- Components 101: Cache, Load Balancer, Database basics
- Study: How web apps scale (relate to your Django/FastAPI experience)

**Communication Practice:**

- Think-aloud while solving (record yourself)
- Goal: Verbalize approach before coding

---

### Week 2: Hash Tables & Searching

**DSA Patterns:**

- Hash Map patterns
- Binary Search variations
- Two Sum family problems

**Daily Flow:**

- 2-3 problems focused on pattern recognition
- Start reflection log: "What would trigger this pattern?"

**System Design:**

- Database choices (SQL vs NoSQL)
- When to cache, what to cache

**Communication Practice:**

- Explain trade-offs out loud
- Practice: "I'm choosing X because..."

---

### Week 3: Stack, Queue, Recursion

**DSA Patterns:**

- Monotonic Stack
- Queue patterns
- Recursion vs Iteration thinking

**Daily Flow:**

- 2 problems + 1 review of past mistakes
- Build pattern trigger cheatsheet

**System Design:**

- Message queues (relate to async processing in your apps)
- When to use background tasks

**Communication Practice:**

- Clarification questions framework
- Never start coding without asking 3 questions

---

### Week 4: Trees & Graphs Introduction

**DSA Patterns:**

- Tree traversals (DFS, BFS)
- Basic graph traversal
- Level-order thinking

**Daily Flow:**

- 2-3 problems (trees are high-frequency)
- Month 1 review: All patterns learned

**System Design:**

- Design a URL shortener (first complete system)
- Practice explaining design choices

**Communication Practice:**

- Mock interview with Claude (30 min DSA)
- Focus: Handling "I don't know" gracefully

**Month 1 Goal:** Recognize 80% of array/string/basic tree patterns instantly

---

## Month 2: Depth & System Thinking (Weeks 5-8)

### Week 5: Advanced Trees

**DSA Patterns:**

- BST operations
- Tree DP patterns
- Lowest Common Ancestor family

**System Design:**

- Design Instagram/Twitter feed
- Introduce: CAP theorem, consistency models

**New Element:**

- Saturday mock: 1 hour (2 DSA + 1 system design question)

---

### Week 6: Dynamic Programming Foundation

**DSA Patterns:**

- 1D DP (climbing stairs, house robber)
- Decision making patterns
- Memoization vs tabulation

**Focus:** This week goes SLOW. DP needs mental model shift.

**System Design:**

- Design notification system
- Focus: Reliability, scale, latency

**Communication:**

- Practice admitting uncertainty: "I'd approach it this way, but let me verify..."

---

### Week 7: Graphs Deep Dive

**DSA Patterns:**

- Advanced DFS/BFS
- Union Find
- Topological Sort

**System Design:**

- Design rate limiter
- Design distributed cache

**Communication:**

- Stretch thinking exercise: For each solution, ask "What if 10x scale?"

---

### Week 8: Advanced DP + Review

**DSA Patterns:**

- 2D DP
- Subsequence problems
- Knapsack variations

**System Design:**

- Design YouTube/Netflix
- Full 45-min design with Claude as interviewer

**Month 2 Goal:** Solve 70% of medium problems correctly on first try

---

## Month 3: Interview Excellence (Weeks 9-12)

### Week 9: Hard Problems + Edge Cases

**Focus:** Combination problems, optimization
**System Design:** Design Uber/food delivery
**New:** Company-specific problem patterns (Google/Microsoft style)

---

### Week 10: Speed & Communication

**DSA:** Timed problems (40 min each, like real interviews)
**System Design:** Design search engine / distributed storage
**Practice:** Full mock interviews 3x this week

---

### Week 11: Weak Spot Elimination

**Review all reflection logs** - identify recurring struggles
**Targeted practice** on identified gaps
**System Design:** Review all past designs, optimize them

---

### Week 12: Interview Simulation Week

**Monday-Friday:** Daily full mock interviews (DSA + System Design)
**Weekend:** Company research, behavioral prep
**Goal:** Feel bored by interview format (you've done it 20+ times)

---

## Success Metrics by Month

**Month 1:**

- Recognize 10 core patterns instantly
- Solve 60% of medium problems in 30 min
- Explain one complete system design

**Month 2:**

- Solve 70% of medium problems in 25 min
- Design 5 systems confidently
- Communicate approach clearly in mocks

**Month 3:**

- Solve 80% of medium + 30% of hard problems
- Handle system design follow-ups smoothly
- Zero freezing during uncertainty

**Final Week:** You're interview-ready for Google/Microsoft
